# HB 1956 - Incarcerated individuals/PRA

*Status: Passed final passage; yeas, 55; nays, 43; absent, 0; excused, 0.* | [leg.wa.gov summary](https://app.leg.wa.gov/billsummary?BillNumber=1956&Year=2021)
