# SB 5010 - Creating an election security camera grant program.

[Source](http://lawfilesext.leg.wa.gov/biennium/2025-26/Pdf/Bills/Senate%20Bills/5010.pdf)

## Section 1
The secretary of state shall create and administer a grant program to purchase and install cameras around ballot drop boxes. Only county auditors and elections directors are eligible for this grant program. Grant funds must only be used for the purchase, installation, or connection of cameras at ballot drop boxes. Grant funds may not be used for ongoing operational costs for cameras.
