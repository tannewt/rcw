# HB 1580 - COVID-19/county phases
**Primary Sponsor:** [Kelly Chambers](/person/leg/chambers_ke.md)

*Status: By resolution, reintroduced and retained in present status.* | [leg.wa.gov summary](https://app.leg.wa.gov/billsummary?BillNumber=1580&Year=2021)
