# SB 5263 - Personal injury defenses
**Primary Sponsor:** [David Frockt](/person/leg/david.frockt.md)

*Status: Effective date 7/25/2021.* | [leg.wa.gov summary](https://app.leg.wa.gov/billsummary?BillNumber=5263&Year=2021)

Concerning defenses in personal injury and wrongful death actions where the person injured or killed was committing a felony.
