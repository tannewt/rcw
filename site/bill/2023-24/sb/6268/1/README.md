# SB 6268 - Agency comms font size

[Source](http://lawfilesext.leg.wa.gov/biennium/2023-24/Pdf/Bills/Senate%20Bills/6268.pdf)

## Section 1
Any communication distributed to the public by a state agency, as defined in RCW 42.56.010, including but not limited to letters, notices, citations, flyers, postcards, and newsletters, must use a minimum of 10 point type and a recommended accessible font, such as Times New Roman, Verdana, Arial, Tahoma, Helvetica, or Calibri fonts. If needed, eight point type is acceptable for tables.
