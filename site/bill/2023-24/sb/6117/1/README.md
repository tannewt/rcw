# SB 6117 - State stone

[Source](http://lawfilesext.leg.wa.gov/biennium/2023-24/Pdf/Bills/Senate%20Bills/6117.pdf)

## Section 1
The Wilkeson Sandstone is hereby designated as the official stone of the state of Washington.
