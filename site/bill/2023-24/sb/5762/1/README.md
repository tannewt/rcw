# SB 5762 - State berry, blueberry

[Source](http://lawfilesext.leg.wa.gov/biennium/2023-24/Pdf/Bills/Senate%20Bills/5762.pdf)

## Section 1
The blueberry is hereby designated as the official berry of the state of Washington.
